## 项目介绍
基于ssm的就业信息平台管理系统：前端 jsp、jquery、jqueryui，后端 springmvc、spring、mybatis，角色分为管理员、企业、用户；集成招聘信息、投递简历、在线企业、新闻资讯等功能于一体的系统。
